Introducción
=============

**Global Exchange Webapp** es una solución empresarial completa para la gestión de casas de cambio y operaciones de intercambio de divisas.

Objetivo
--------

El objetivo principal de Global Exchange Webapp es proporcionar una plataforma robusta y segura para:

- Gestionar operaciones de compra y venta de divisas en tiempo real.
- Administrar clientes con diferentes categorías y límites de transacción.
- Controlar cotizaciones y tipos de cambio con historial completo.
- Procesar transacciones con múltiples métodos de pago y cobro.
- Generar reportes detallados para auditoría y análisis financiero.
- Integrar terminales físicas (T-ausers) para operaciones en punto de venta.

Audiencia
---------

Esta documentación está dirigida a:

- **Administradores del sistema**: responsables de la configuración, gestión de usuarios y mantenimiento.
- **Operadores de caja**: personal que realiza operaciones diarias de compra/venta.
- **Supervisores**: encargados de monitorear operaciones, aprobar transacciones y generar reportes.
- **Desarrolladores**: técnicos que necesiten extender, personalizar o integrar el sistema.
