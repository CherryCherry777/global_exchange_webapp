Introducción
=============

Breve introducción al proyecto Global Exchange Webapp.

Objetivo
--------

Global Exchange Webapp es una aplicación para gestionar intercambios de divisas, clientes, transacciones y reportes administrativos.

Audiencia
--------

Esta documentación está dirigida a desarrolladores y administradores del sistema que necesiten entender la arquitectura, workflows y cómo operar la plataforma.

Estructura de la documentación
------------------------------

- Sección 1: Primeros pasos — instrucciones para instalar y empezar a usar el sistema.
- Sección 2: Módulos — documentación por módulo (modelos, vistas, formularios).
- Sección 3: Operaciones — guías operativas y procedimientos de soporte.

Notas
-----

Si necesitas que añada un glosario o mapas de rutas (diagrama de navegación), indícamelo y lo incluyo.
