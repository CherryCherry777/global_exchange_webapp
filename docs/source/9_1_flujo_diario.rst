9.1 Flujo típico de operación diaria
===================================

Resumen
------

Descripción del flujo diario recomendado para operadores y administradores.

Pasos clave
-----------

1. Verificar el dashboard de inicio y alertas (pendientes, errores, terminales offline).
2. Revisar clientes con tareas pendientes (kyc, límites a revisar).
3. Procesar órdenes de compra/venta siguiendo el flujo de confirmación y conciliación.
4. Ejecutar cierres parciales si es necesario y registrar arqueos en terminales.

Tareas de cierre de día
-----------------------

- Generar reportes diarios y revisar discrepancias en límites y saldos.
- Confirmar que no queden transacciones en estado intermedio.

Consejos operativos
-------------------

- Documentar excepciones y problemas para facilitar soporte y auditoría.
- Usar exportaciones CSV para conciliaciones contables si se requiere.
