7.3 Monitoreo de T-ausers
=========================

Indicadores clave
-----------------

- Estado (online / offline)
- Última sincronización
- Número de transacciones procesadas
- Alarmas por errores repetidos o desconexiones

Herramientas y vistas
---------------------

- Dashboard de terminales: lista y filtros por estado y ubicación.
- Logs y trazas: seguimiento de peticiones y respuestas desde cada terminal.

Alertas y acciones
------------------

- Configurar alertas por terminal offline > X minutos.
- Procedimientos para reinicio remoto o bloqueo de terminal comprometido.
