Descripción
-----------

Este glosario proporciona definiciones claras de los términos técnicos, conceptos de negocio y abreviaturas utilizadas en la documentación y el código del sistema Global Exchange Webapp.

Organización
------------

**Términos técnicos**

Definiciones de conceptos técnicos y de negocio utilizados en el sistema:

- Conceptos financieros (cotización, spread, transacción).
- Entidades del sistema (cliente, moneda, billetera).
- Conceptos operativos (arqueo, límite, denominación).
- Términos técnicos (API, webhook, autenticación).

**Abreviaturas**

Lista de acrónimos y siglas utilizadas:

- Abreviaturas técnicas (API, DB, CLI).
- Abreviaturas de industria (FX, KYC, AML).
- Abreviaturas de uso común (CRUD, CSV, UTC).

Cómo usar este glosario
-----------------------

- Consulta este glosario cuando encuentres un término desconocido en la documentación.
- Los términos están ordenados alfabéticamente dentro de cada sección.
- Si encuentras un término que debería estar incluido, reporta un issue para agregarlo.
