Navegación básica
=================

Esta guía describe la estructura de la interfaz de usuario y cómo navegar eficientemente por las diferentes secciones del sistema.

Estructura general de la interfaz
---------------------------------

La interfaz de Global Exchange Webapp está organizada en las siguientes áreas principales:

**Dashboard / Inicio**

- Panel principal que muestra métricas clave del día: volumen de transacciones, ingresos por comisiones, alertas pendientes.
- Widgets configurables según el rol del usuario.
- Accesos rápidos a las operaciones más frecuentes.

**Clientes**

- Listado completo de clientes registrados en el sistema.
- Funciones de búsqueda, filtrado y ordenamiento.
- Acceso a la ficha individual de cada cliente con su historial.

**Transacciones**

- Formulario para crear nuevas operaciones de compra/venta.
- Historial de transacciones realizadas con detalles completos.
- Estados de las transacciones: pendiente, completada, cancelada.

**Reportes**

- Generación de informes por periodo, cliente, moneda o categoría.
- Exportación en formatos CSV y PDF.
- Métricas de rendimiento y análisis de tendencias.

**Configuración / Administración**

- Gestión de usuarios, roles y permisos.
- Configuración de monedas, cotizaciones y límites.
- Parámetros del sistema y preferencias.

Barra de navegación superior
----------------------------

La barra superior contiene elementos de acceso rápido:

- **Logo/Inicio**: Clic para volver al dashboard principal.
- **Menú de usuario**: Desplegable con opciones de perfil, preferencias y cerrar sesión.
- **Notificaciones**: Campana con indicador de alertas pendientes.
- **Búsqueda global**: Campo para buscar clientes, transacciones o configuraciones.

Menú lateral de navegación
--------------------------

El menú lateral izquierdo organiza las funcionalidades por categoría:

- **Operaciones**: Compra/Venta, Historial, Cotizaciones.
- **Clientes**: Listado, Nuevo cliente, Categorías, Asignaciones.
- **Reportes**: Transacciones, Comisiones, Límites consumidos.
- **Terminales**: T-ausers, Denominaciones, Arqueos.
- **Configuración**: Usuarios, Roles, Monedas, Métodos de pago, Parámetros.

El menú puede colapsarse para ganar espacio de trabajo en pantallas pequeñas.

Búsqueda y filtros avanzados
----------------------------

**Búsqueda por texto:**

- Los listados incluyen un campo de búsqueda que filtra en tiempo real.
- La búsqueda opera sobre campos clave: nombre, documento, código de transacción.

**Filtros disponibles:**

- **Por fecha**: Seleccionar rango de fechas con calendario interactivo.
- **Por categoría**: Filtrar clientes según su categoría asignada.
- **Por estado**: Mostrar solo registros activos, inactivos, pendientes, etc.
- **Por moneda**: Filtrar transacciones por moneda origen o destino.

**Ordenamiento:**

- Hacer clic en los encabezados de columna para ordenar ascendente/descendente.
- El sistema recuerda el último ordenamiento aplicado.

Acciones en masa
----------------

- Seleccionar múltiples registros usando las casillas de verificación.
- Aplicar acciones como: activar, desactivar, exportar selección, eliminar.
- Las acciones en masa requieren confirmación para operaciones destructivas.

Atajos de teclado
-----------------

- ``Ctrl + N``: Crear nuevo registro (cliente, transacción).
- ``Ctrl + F``: Enfocar campo de búsqueda.
- ``Esc``: Cerrar diálogos y modales.

Recomendaciones de uso
----------------------

- Utilizar las vistas de gestión (``manage_*``) como punto de entrada principal para cada módulo.
- Aprovechar la exportación CSV para análisis detallados en hojas de cálculo.
- Revisar el dashboard al inicio de cada jornada para identificar alertas y tareas pendientes.
- Usar favoritos del navegador para acceder rápidamente a las secciones más utilizadas.
